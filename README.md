# **Tiger Paw Money**

- Private bookkeeping app to track your spending  
- Supports Simplified Chinese (RMB), English (USD), and Korean (KRW)  
- Professional Excel-style UI with intelligent natural week statistics  

## **Core Features**

- **Excel-Style High-Density UI (NEW)**  
  - Optimized for productivity with a compact grid layout  
  - View dozens of transactions on a single screen without excessive scrolling  

- **Full-Field Instant Edit (NEW)**  
  - Click directly on any cell (Date, Category, City, Note, Amount) to edit  
  - No complex menus or extra steps  

- **Category Breakdown Pie Chart (NEW)**  
  - Visualizes spending distribution clearly  
  - Percentage labels remain visible when a date range is selected  

- **Natural Week Statistics (REFINED)**  
  - Automatically tracks spending from Monday to Sunday  
  - Matches real-world weekly habits  

- **Multi-Currency Engine**  
  - Seamless conversion between CNY, USD, and KRW  
  - Real-time exchange rate synchronization  

- **Performance Optimized**  
  - Smooth handling of 1000+ entries  
  - Asynchronous rendering for fast interactions  

## **Privacy & Data Storage**

- **Pure Local Storage (LocalStorage)**  
  - All financial data stays on your device  
  - GitHub only hosts the interface and cannot access your data  

- **Security**  
  - Data is device-specific  
  - Different users opening the same URL only see their own records  

- **Backup Mechanism**  
  - Export data as a JSON file using the Export feature  
  - Restore data with Import when switching devices  

## **Quick Start(exciting!!!!)**

- **Access**  
  - Open the deployed GitHub Pages URL  
  - Or download `index.html` and run locally  

- **Log**  
  - Enter details and click Log  
  - Custom categories and cities are remembered automatically  

- **Analyze**  
  - Select a date range to activate the Category Pie Chart
